# S0 — 랜딩 페이지 (ASCII 와이어프레임)

## 전제

- 세로 스크롤 페이지. 데스크톱 폭 1440 가정.
- **싱클리 정보 구조 패턴 5개 흡수** + DILAB 의도적 차별화 4개 반영
- 친숙 톤(v2): "당신에게?" 직접 어필 / 이모지 라벨 1개씩 / 부정 = 회색

## 페이지 구조 (11블록, 세로 흐름)

```
┌──────────────────────────────────────────────────────────────────────────────┐
│  HERO       ▷  히어로 / 가치 제안 / 1차+2차 CTA                              │
├──────────────────────────────────────────────────────────────────────────────┤
│  TRUST      ▷  학술 신뢰 신호 + 출처 카드 미리보기 (DILAB 식 차별화)        │
├──────────────────────────────────────────────────────────────────────────────┤
│  WHEN       ▷  활용 시나리오 3개 (When > What)                              │
├──────────────────────────────────────────────────────────────────────────────┤
│  PAIN x SOL ▷  페인 3개 ↔ 솔루션 3개 1:1 매트릭스                            │
├──────────────────────────────────────────────────────────────────────────────┤
│  DEMO       ▷  DILAB Ask 라이브 데모 (출처 카드 가시화 — 외부 노출)         │
├──────────────────────────────────────────────────────────────────────────────┤
│  WORKFLOW   ▷  내부 동작 4단계 (① 수집 → ② 분석 → ③ 검색 → ④ 출처 명시)     │
├──────────────────────────────────────────────────────────────────────────────┤
│  SHOWCASE   ▷  실제 분석 리포트 쇼케이스 2개 (S1 진입)                      │
├──────────────────────────────────────────────────────────────────────────────┤
│  EXPERT     ▷  전문가 인용 3개 (이름·전문 분야·신뢰도)                       │
├──────────────────────────────────────────────────────────────────────────────┤
│  PRICING    ▷  3티어 카드 (Free / Pro / Custom) + 자가 가입 강조             │
├──────────────────────────────────────────────────────────────────────────────┤
│  FAQ        ▷  통합 FAQ 4개 (별도 헬프 X)                                    │
├──────────────────────────────────────────────────────────────────────────────┤
│  CTA + 풋터                                                                  │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 블록 1 — HERO

```
┌──────────────────────────────────────────────────────────────────────────────┐
│  DILAB                                       제품 ▼  도메인 ▼  가격  로그인  │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│       리뷰가 아무리 많아도,                                                  │
│       전문가 한 마디가 없으면 결정이 어렵죠.                                 │
│                                                                              │
│       DILAB 은 도메인 전문가 리뷰 + 일반 리뷰를 함께 분석해                  │
│       *어디서 가져온 결론인지* 매번 보여주는 제품 평가 도구예요.             │
│                                                                              │
│       [▶ 무료로 시작하기]     [🔍 데모 리포트 보기]                          │
│                                                                              │
│       ──────────────────────                                                 │
│       ⓘ 가입 시 카드 등록 불필요 · 월 20건 무료                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

**핵심**:
- 헤드라인이 *페인 → 가치 한 줄* 구조. "리뷰 많아도" (인지된 문제) → "전문가 한 마디" (DILAB 의 답)
- 1차 CTA: **"무료로 시작하기"** ← 싱클리의 "도입 문의" 와 정반대 (차별-1)
- 2차 CTA: "데모 리포트 보기" — S1 으로 바로 이동, 영업 컨택 없이 *제품 가치* 먼저 보여줌
- CTA 아래 "카드 등록 불필요" 자가 가입 강조 마이크로카피

- 학술 신뢰 신호: "임상순 교수님 연구실 + 한국어 도메인 RAG 연구" — 큰 로고는 없지만 *연구 정체성* 으로 권위 부여
